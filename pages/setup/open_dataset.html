Test the setup worked!#

To-do

Make sure you are connected to the WHOI CMIP server. (See instructions in previous section.)

Activate your virtual environment and start a jupyterlab session. For example, if your mamba/conda environment is called my_new_env, enter the following lines in a command window (the first line activates the virtual environment and the second line starts the jupyter session):

> conda activate my_new_env
> jupyter lab

Then, execute the cells in the rest of this notebook (you can either copy/paste code to a blank notebook, or download this notebook directly).

Open a dataset using xarray#

To-do

Choose ONE of the following four datasets from the WHOI CMIP server to open, or adapt the code to open a different dataset of your choosing. Please do not open more than one of the four example datsets, as the server gets overwhelmed when too many people are trying to access the same dataset at the same time. If you want to choose your own dataset, the following website is helpful for finding one and getting the filepath: http://cmip6.whoi.edu/search/.

Option 4: Ocean Current Velocity in ECCO Ocean Reanalysis#

# U velocity
path = pathlib.Path('/Volumes/cmip6/data/ocean_reanalysis/ECCOv5/Version5/Alpha/latlon_monthly/UVEL.nc')
files = list(path.glob("*.nc"))
U_ds = xr.open_mfdataset(files[0:12], decode_times=False).UVEL

# V velocity
path = pathlib.Path('/Volumes/cmip6/data/ocean_reanalysis/ECCOv5/Version5/Alpha/latlon_monthly/VVEL.nc')
files = list(path.glob("*.nc"))
V_ds = xr.open_mfdataset(files[0:12], decode_times=False).VVEL

# Manually decode TIME
start_date = pd.Timestamp("1991-12-01")
months_since = U_ds["TIME"].values
U_ds['TIME'] = [start_date + pd.DateOffset(months=int(m)) for m in months_since]
V_ds['TIME'] = [start_date + pd.DateOffset(months=int(m)) for m in months_since]

# Combine U and V variables into one dataset using xarray
ds = xr.merge([U_ds, V_ds])
