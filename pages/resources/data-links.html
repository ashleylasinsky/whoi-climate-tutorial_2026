Data links#

List of observational/reanalysis climate datasets#

Most of these sites provide access to actual data files (e.g., in NetCDF format). Some also include a web tool to make quick plots and/or subsets of the data.

  • NOAA PSD gridded climate data sets

  • NOAA NCDC online climate data

  • NOAA Paleoclimatology data center

  • NCAR/UCAR Climate Data Guide

  • Reanalysis intercomparison

  • UK Met Office Hadley Centre observational data sets

  • CESM-MARBL-FEISTY

  • MERRA-NOBM (NASA Ocean Biogeochemical Model) Reanalysis

Other useful resources#

  • Atmospheric reanalysis overview/comparison: provided by UCAR; scroll down to bottom of page for summary table

  • Climate Variability and Diagnostics Package (CVDP): data repository containing summary statistics and pre-computed climate indices (e.g., Niño 3.4, AMOC index, etc) for CMIP models.
